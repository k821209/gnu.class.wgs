PacBio SMRT 시퀀싱 기술

📡 Single Molecule Real Time (SMRT) 시퀀싱

🔬 SMRT 시퀀싱 원리

  1. SMRT 셀 준비: 나노크기 웰(ZMW, Zero-Mode Waveguide) 배열
  2. 단일분자 로딩: 각 웰에 하나의 DNA 중합효소와 템플릿
  3. 실시간 합성: DNA 중합효소가 형광 표지 뉴클레오타이드 첨가
  4. 형광 검출: 각 염기 첨가 시 실시간 형광 신호 감지
  5. 연속 읽기: 중합효소가 계속 진행하며 긴 읽기 생성

⚙️ Zero-Mode Waveguide (ZMW) 기술

  • 초소형 웰: 직경 ~70nm, 깊이 ~100nm
  • 광학적 격리: 각 웰이 독립적인 광학 챔버 역할
  • 배경 노이즈 차단: 웰 밖 형광 신호 차단
  • 단일분자 검출: 극소량 DNA에서도 신호 검출

Zero-Mode Waveguide Technology Explained
PacBio SMRT 시퀀싱의 핵심 기술인 ZMW 웰의 작동 원리와 단일분자 검출 메커니즘
🔬 ZMW 기술의 혁신 포인트:
• 나노포토닉스: 빛의 회절 한계를 극복한 나노 광학 기술
• 제로모드 효과: 웰 크기가 여기광 파장보다 작아 빛 침투 제한
• 아토리터 부피: 10⁻¹⁸ L 극소 반응 공간에서 단일분자 관찰
• 실시간 관찰: DNA 중합효소 활동을 실시간으로 직접 관찰
• 장거리 읽기: 중합효소가 계속 작동하는 동안 연속 시퀀싱

🎯 PacBio HiFi (High Fidelity) 기술

🔄 Circular Consensus Sequencing (CCS)

  • 원형 템플릿: SMRTbell 어댑터로 원형 구조 형성
  • 다중 통과: 동일 분자를 여러 번 시퀀싱
  • 합의 서열: 여러 읽기에서 합의 서열 생성
  • 오류 보정: 무작위 오류는 상쇄, 진짜 변이는 보존
📹 CCS 기술 시각화:
Circular Consensus Sequencing Explained
SMRTbell 어댑터와 다중 통과 시퀀싱으로 정확도를 높이는 CCS 기술의 작동 원리
🔄 CCS 핵심 개념:
• SMRTbell: 원형 이중가닥 DNA 구조
• 다중 패스: 동일 분자 3-15회 반복 읽기
• 합의 알고리즘: 다수결 원리로 오류 제거
• Q값 향상: 통과 횟수 증가 시 정확도 지수적 증가

📊 HiFi 성능 지표

  • 정확도: >99.8% (Q30 이상)
  • 읽기 길이: 평균 10-25kb
  • 처리량: Revio 시스템 최대 1.2Tb
  • 균일성: 고른 커버리지 분포

🏭 PacBio 플랫폼

  • Revio: 최신 고처리량 long-read 시스템
  • Sequel IIe: 중간 처리량 long-read 시스템
  • Onso: 정확도 중심 short-read 시스템 (SBS 기술)
  • 통합 워크플로우: SMRT Link 소프트웨어
📝 참고: PacBio Onso는 2022년 출시된 short-read 시퀀서로,
Sequencing by Synthesis (SBS) 기술을 사용하며 최대 2×150bp 읽기 길이와
>99.9% 정확도를 제공합니다. 이는 PacBio의 유일한 short-read 제품입니다.
📹 Revio 시스템 소개:
PacBio Revio System Overview
차세대 고처리량 HiFi 시퀀싱 플랫폼 Revio의 혁신적 기술과 성능
🚀 Revio 혁신 특징:
• 초고처리량: 런당 1.2Tb 데이터 생산
• 25M ZMW: 2,500만 개 나노웰 동시 시퀀싱
• 15시간 런타임: 빠른 턴어라운드 시간
• 비용 효율성: 30× 인간 게놈 $1000 달성
• 자동화: 통합 워크플로우 및 데이터 관리
📊 Revio 시스템 사양 (2024 공식 스펙):
• SMRT Cell 구성: 4개 SMRT Cell 25M (2,500만 ZMW) 동시 런
• 표준 셀당 출력: 90-100Gb HiFi data per SMRT Cell
• SPRQ 개선 출력: 100-120Gb per SMRT Cell (33% 증가)
• 최대 검증 출력: 110.5Gb per SMRT Cell (실제 달성값)
• 시스템 총 처리량: 런당 최대 480Gb (4셀 × 120Gb)
• 플렉서블 운영: 1-4개 셀 부분 런 지원
🧮 실제 게놈 처리량 계산:
• 인간 게놈 크기: ~3.2Gb (30× 커버리지 = 96Gb 필요)
• 셀당 게놈 수: 100-120Gb ÷ 96Gb = 1-1.25개 게놈
• 런당 게놈 수: 4셀 × 1.25 = 최대 5개 게놈 (30×)
• 낮은 커버리지: 20× 커버리지 시 셀당 1.5-2개 게놈
• 연간 처리량: 주 1회 런 시 연간 ~250-300 게놈
💰 SPRQ 화학법 개선 (2024년 10월):
• 데이터 증가: 90Gb → 120Gb (33% 향상)
• DNA 입력량 감소: 2000ng → 500ng (4배 절약)
• 비용 절감: 인간 게놈당 $1000 → $500
• 처리량 향상: 셀당 더 많은 샘플 처리 가능

✅ PacBio HiFi의 고유 장점

  • Short-read 수준 정확도: 긴 읽기 + 높은 정확도 동시 달성
  • 균등한 커버리지: GC bias 최소화
  • 메틸화 검출: 5mC, 6mA 등 직접 검출
  • 구조변이 민감도: 50bp 이상 SV 95% 이상 검출
